# Pi Extensions

A few of the Pi extensions I've built for myself.

## Extensions

| Extension | Description |
| --- | --- |
| [`fast-mode`](extensions/fast-mode) | Adds a `/fast` toggle for supported Codex/OpenAI GPT-5.4/GPT-5.5 models (service_tier=priority). |
| [`iterate`](extensions/iterate) | Runs parallel Pi agents in isolated git worktrees so you can compare and merge the best result. |
| [`pi-goal`](extensions/pi-goal) | Adds a Codex-style persistent `/goal` command that keeps working until the objective is complete. |
| [`computer-use`](extensions/computer-use) | Exposes Codex-installed macOS Computer Use tools to any Pi model. |
